Firebase bilgilerini gir, uygulama tüm cihazlarda gerçek zamanlı çalışsın.

1Firebase projesi oluştur

console.firebase.google.com → "Add project" → proje adı yaz → Continue → Continue → Create project

2Realtime Database ekle

Sol menü → Build → Realtime Database → Create database → Start in test mode → Enable

3Web app ekle ve config al

Project Settings (⚙️) → General → "Add app" → Web (</>) → Register app → aşağıdaki değerleri kopyala
